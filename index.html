System Architecture

Under the hood of the v12.3 Enterprise Master engine.

01 // API_ROUTING

Centralized Webhook Router

Unified doGet and doPost architecture for managing tracking pixels, public subscriptions, and administrative COMPOSE requests.

02 // TRACKING_ENGINE

Invisible Pixel Engine

Integrated 1x1 tracking pixel generator with real-time open/click synchronization directly to your subscriber nodes.

03 // GMAIL_SYNC

Gmail History Engine

Real-time retrieval of Gmail threads for any subscriber ID, providing full conversation context within the admin console.

04 // RESOURCE_MGMT

Daily Quota Sensor

Active monitoring of MailApp.getRemainingDailyQuota() to prevent delivery failures and manage system throughput.

05 // DATA_INTEGRITY

V8 Concurrency Lock

Uses LockService to prevent spreadsheet corruption during high-concurrency write events and API surges.

06 // AUTH_LAYER

Multi-Key Security

Protected by a Secret Master Key and User Session Tokens with automated brute-force lockout protection (5-attempt limit).
